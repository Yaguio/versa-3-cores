# Versa 3 Cores

Um desafio de 15 dias: Hernalton Junior, motorista de app da Baixada Fluminense, precisa fechar a grana suficiente pra comprar tudo que acha importante — do bolo no sinal até a pensão atrasada — driblando família, cansaço, fome, o agiota CP e as ruas do Rio, sem fundir o motor ou quebrar de vez.

Jogo single-file (HTML + CSS + JS puro, sem build, sem dependências, sem servidor). Abra `index.html` em qualquer navegador pra jogar.

## Jogar localmente

Basta abrir `index.html` direto no navegador, ou servir a pasta com qualquer servidor estático:

```bash
npx serve .
```
