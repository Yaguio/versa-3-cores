# Versa 3 Cores

Um desafio de 30 dias: Hernalton Junior, motorista de app da Baixada Fluminense, precisa juntar dinheiro suficiente pra comprar tudo que acha importante — do bolo no sinal até a pensão do mês — driblando família, fadiga, fome, o agiota CP e as ruas do Rio, sem fundir o motor ou falir.

Jogo single-file (HTML + CSS + JS puro, sem build, sem dependências, sem servidor). Abra `index.html` em qualquer navegador pra jogar.

## Jogar localmente

Basta abrir `index.html` direto no navegador, ou servir a pasta com qualquer servidor estático:

```bash
npx serve .
```
